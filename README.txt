== Description ==

 A simple plugin designed specifically to explain the structure of the template based on the principles of MVC 

== Installation ==


 1. Upload `my-plugin` to the `/wp-content/plugins/` directory
 2. Activate the plugin through the 'Plugins' menu in WordPress
 3. Configure the plugin in the menu "Wp Plugin"
